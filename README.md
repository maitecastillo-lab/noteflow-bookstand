# NoteFlow
App móvil para gestionar **notas, tareas e ideas** en un solo sitio.

---

## Qué hace

Tres tipos de contenido con su propio diseño y comportamiento:

- **Notas** — texto libre con título y contenido
- **Tareas** — listas con checkboxes y barra de progreso
- **Ideas** — apuntes rápidos con etiquetas y colores

Y lo de siempre: crear, editar, archivar, eliminar y buscar. Modo claro y oscuro. Todo se guarda solo en el móvil.

---
## Pantallas

- **Notas / Tareas / Ideas** — cada una con su lista y su botón de crear
- **Archivadas** — lo que has guardado sin borrar, con opción de recuperar o eliminar definitivamente
- **Ajustes** — cambiar entre claro/oscuro y ver estadísticas

---
## Stack

- **Expo + TypeScript**
- **Expo Router** para navegación
- **Gluestack UI** para el diseño
- **Zustand + AsyncStorage** para estado y persistencia
- **FlashList** para listas rápidas
- **Zod** para validar formularios
- **Reanimated** para animaciones
- **expo-haptics** para vibraciones
